App Name: RouteWeather
Purpose: Project weather forecast onto travel route using Google Maps API, in order to plan travel

How to use:

Short press - add vertice on route
Long press - open UI for entering forecast and erasing points
Slider bar in UI - slide to choose how far ahead to give forecast

After submitting form, if there will be rain, then there will be markers displayed. Click the marker to show precipitation. If there are no markers then it is not predicted to rain.

Weather API source: https://openweathermap.org/api/one-call-api#data

Planned features:
